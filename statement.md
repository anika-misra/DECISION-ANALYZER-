# PROBLEM STATEMENT
People often struggle to make decisions as they cannot properly compare and analyze the best option for them. Evaluating the pros and cons of the option chosen is a difficult task for them . This Decision Analyzer makes it easy to choose the option which is feasible and does not involve any risks.

# SCOPE OF VARIABLE
1. Variable Name: Situations
2. Function: It analyzes the situation and gives the most appropriate option .
3. Limitation: It can not give more suggestions and only works with the situation which is pre defined and just gives the suitable output.

# TARGETED USER
1. Young minds to elderly .
2. Confused people.
3. Poor decision makers.

# HIGH-LEVEL FEATURES
1. Command interface.
2. Instant response generation.
3. Easy and interactive interface.
4. Persistent data storage.
