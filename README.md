# DECISION-ANALYZER
 
# OBJECTIVE
The main objective of decision analyzer is to help people make complex day to day decisions by combining the data provided in the form of situation and with the help of adaptive reasoning. It evaluates multiple options and gives the most suitable option. 

# FEATURES
1. Sentiment Analysis : Analyses the options on the basis of positive(e.g., benifit, easy, opportunity)  and negative(e.g., risks, uncertainity, difficulty) aspects.
2. Clarity Scoring : Favors concise and necessary or urgent decisions first.
3. Risk Adjustment : Eliminates out the option with high risk .
4. Transparent Output : Provides a detailed reason breakdown of the option chosen .

# TOOLS USED 
1. Language - Python
2. Editor - Python IDLE
3. References - Modules, Academic papers and Online sites

# STEPS TO INSTALL AND RUN THE PROJECT 
1. Install PHYTHON
2. Open a new file in IDLE Shell.
3. Create a file name "decision-analyzer.py".
4. Write code and save it .
5. Execute code .
6. Enter both the situations and get the output.

# INSTRUCTIONS FOR TESTING
1. Start the program.
2. Enter both the situations .
3. The analyzer will analyze both the situations and  give the best suitable option.
4. The program would end after giving the output.
5. For another situation , run the program again.
